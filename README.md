# 数据挖掘、数据分析及可视化

## 要求：



- [x] 通过互联网收集企业的数据，主要包括企业名字、成立时间、企业规模、企业简历、发展历程、企业类别、联系方式、主要产品和应用案例等信息。
- [x] 再使用大数据分析方法对数据进行清洗和标准化。
- [x] 接着使用数据挖掘和机器学习的方法对数据进行聚类，把具有相同特征的企业划分到同一个类别里。
- [x] 最后对不同类别的企业进行分析。
- [x] 把分析的结果进行可视化。



爬取数据对象就是长春市入户科技型中小企业
